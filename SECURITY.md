# Security Policy

## Supported Versions

This project is under active development. The latest version in the main branch is considered supported.

Older versions may not receive updates or security fixes.

---

## Reporting a Vulnerability

If you discover a security issue or vulnerability in this project, please report it responsibly.

- Do NOT open a public issue for sensitive vulnerabilities
- Contact me directly via GitHub (create a private report if possible)

Please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fixes (if available)

---

## Scope

This project performs static analysis only and does not execute binaries. However:

- Always use this tool in a safe, isolated environment (VM or sandbox)
- Do not analyze untrusted files on your host system
- Treat all samples as potentially malicious
